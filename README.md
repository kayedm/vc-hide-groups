<div align="center">

A Vencord plugin that allows you to remove groups from your direct messages list on Discord.  

<img src="assets/demo.gif" alt="demo"/>

</div>

## Installation

### 1.
Clone the <a href="https://github.com/Vendicated/Vencord">Vencord</a> repo and add `index.tsx` to `src/userplugins/HideGroups` (create the folder if necessary). 

### 2.
Build the app using `npm run build` and `npm run inject` to apply to Discord. 

### 3.
Enable the plugin in settings.
